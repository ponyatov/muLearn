@brief Ищу онлайн-платформу для микрообучения, пока бесплатно для тестов.

Направление крайне перспективное, Усилия должно быть 
легко монетизировать если нацеливаться на рынок mobile first:
платный учебный контент в виде микроюнитов стоимостью 10-50 р
в режиме MOOC с тысячами студентов.

uSecase: юзер в транспорте (метро/автобус/пробки) или на перервыве (в очереди и т.п.), 
юниты со средней длительностью 15-20 минут,
желательно наличие мобильного клиента способного кешировать контент для частых обрывов связи (метро).

Ключевая фишка: максимально привлекать для создания курсов прикладных специалистов,
профессионально работающих в конкретной области, 
поэтому затраты времени на создание юнита должны быть очень небольшими (не более 1-2 часов),
быстро окупающиеся за счет очень низкой стоимости юнита х количество юзеров.

В идеале: создание платформы с бесплатной стоимостью входа, настолько удобной, что
преподавателям (вторая целевая группа) проще готовить лекции в платформе, и использовать
ее в качестве бэка для очных лекций и презентаций.

Другой сегмент рынка: платформа для обучения пользователей технически-сложных продуктов, типа https://icbcom.ru/ru/
с нулевой стоимостью владения, и первоначально бесплатной для компаний размещающих контент.
